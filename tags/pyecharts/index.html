    pyecharts 中的 tooltip_formatter

    近日用 pyecharts 实现关系图（Graph），需要自定义tooltip，即鼠标在关系连线 link 或节点 node 上悬停时的显示内容。 官方文档写得比较抽象，网上搜了下，未能找到可以直接套用的案例。 经过一番理解，最终成功实现了。 笔者使用的 pyecharts 版本为 0.5.11，python 版本为 3.7，现总结如下： 根据官方文档，自定义 tooltip 涉及参数 tooltip_formatter， pyecharts 中的这个参数是在 graph.add 阶段加入的。 //定义回调函数 def tooltip_formatter_des(params): if '>' in params.name: //偷懒通过 '>' 区别 link 和 node for des in des_show: // des_show为预定义数组 temp = des.split('<br>') if temp[0] == params.name: return des else: return params.name graph.add( name, nodes, links, is_label_show=True, graph_repulsion=8000, graph_layout=None, label_text_color=Non line_curve=0.05, graph_edge_symbol=[None, 'arrow'], line_color= 'black', tooltip_formatter= tooltip_formatter_des //关键参数，回调函数名作为参数 ) 实际发现如此配置，仍然无法正确显示 tooltip
